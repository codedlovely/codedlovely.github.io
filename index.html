Hi there

My name is Ai Kawasaki. I'm a self-taught front-end developer based in Japan, currently seeking opportunities to turn my enthusiasm for coding into websites that help people's everyday life.

I found my passion for web development, specifically for HTML/CSS coding, when I started creating my personal website more than 20 years ago. Since then, I've learned HTML/CSS and JavaScript (occasionally PHP and other server side script) on my own, and I'm still enthusiastically absorbing new technical information to expand my skill set.

I also have experience with VS Code, WordPress, Jekyll, Tableau, Presentation prep.

When I'm not coding, I'm running, scuba diving, baking, reading, playing ukulele, singing.
